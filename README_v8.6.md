## 🚀 新功能概览

### 随机间断功能

在运行过程中，程序会随机暂停休息，模拟真实用户的使用习惯：

```
运行 12 分钟 → 休息 3 分钟 → 运行 15 分钟 → 休息 4 分钟 → ...
```

**关键特性：**
- ⏱️ 每隔 10-20 分钟随机休息一次
- 💤 休息时长可自定义（默认 2-5 分钟）
- 📊 休息期间显示倒计时
- 🎯 休息时间不计入总运行时间
- 🔄 完全随机化，增强防检测
